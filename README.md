# Python Django 图书管理系统 python751

## 项目概述

本项目是一个基于Python Django框架开发的图书管理系统，旨在提供一套简洁、易用、功能齐全的图书管理解决方案。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript, AJAX

## 功能模块

- 用户模块：登录、注册、登出、修改密码
- 图书模块：添加图书、查找图书信息
- 作者与出版社模块：添加作者、出版社信息，查找出版社信息
- 数据处理：密码加密、信息唯一性判断

## 系统角色

- 系统管理员：负责系统的维护与管理
- 普通用户：可以查看图书信息，进行借阅等操作

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.6及以上
- 数据库：MySQL 5.7及以上

## 部署步骤

1. 安装Python及其依赖库
2. 安装MySQL数据库，并导入数据
3. 部署至Web服务器，如Nginx或Apache
4. 启动Django应用

## 核心亮点

- 使用Django框架，保证了系统的稳定性与可维护性
- 对用户密码进行加密存储，确保了用户信息安全
- 强调信息唯一性判断，避免了数据的重复与冗余
